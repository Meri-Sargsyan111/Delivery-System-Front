About the System

A microservices-based delivery management platform

Delivery System is a delivery management platform built on a microservices architecture. It coordinates the full lifecycle of a delivery order — from creation through courier processing to final delivery — while keeping each responsibility isolated in its own backend service and communicating through REST APIs and event-driven messaging.

Order Management

Create and manage delivery orders, including their status throughout the delivery lifecycle.

Courier Processing

A dedicated courier service handles courier delivery processing for each order.

Live Location Tracking

Courier positions are tracked in real time and rendered on Google Maps.

Real-Time Communication

WebSocket/STOMP channels stream live courier updates to the frontend as they happen.

Event-Driven Backend

Apache Kafka connects services asynchronously for scalable, decoupled communication.

Tracking History

Delivery status changes are recorded, preserving a history of each order's journey.

Notifications

Users receive real-time notifications about relevant delivery events.

Authentication & Profiles

Secure authentication protects the platform, with user profile management for each account.
